# Design System

## Typography

**Display / UI:** Poppins (Google Fonts, OFL)

Poppins is the canonical typeface for all text in the app, matching the Mac desktop app. Loaded weights:

| Weight | Family name |
|--------|-------------|
| Light | `Poppins-Light` (300) |
| Regular | `Poppins-Regular` (400) |
| Medium | `Poppins-Medium` (500) |
| SemiBold | `Poppins-SemiBold` (600) |
| Bold | `Poppins-Bold` (700) |

**Chrome / controls:** SF Pro (system) via `system-ui` / `ui-*` fallbacks on iOS.

**Monospace:** System monospace (`ui-monospace` on iOS, `monospace` elsewhere) for code spans.

**Long-form copy:** Future consideration — a serif face (Source Serif 4 or similar) may be introduced for article-length reading views. Not implemented in v1.
